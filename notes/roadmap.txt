3. Node-RED + SQLite/MySQL (2 hours)

Install node-red-node-sqlite or the MySQL node. Practice writing flow data into a database table — this maps directly to your existing SQL knowledge.
4. Node-RED + AWS SNS (1–2 hours)

Use the node-red-contrib-aws package or a plain HTTP request node hitting the SNS API. You've already used SNS at ORDI so this will feel familiar.
5. Build the actual pipeline (1–2 days)

Only now wire it all together: simulated sensor input → data transformation → anomaly detection logic → DB logging → SNS alert.
6. Deploy on Linux + document it (half a day)

Run it as a systemd service on an EC2 or local Linux VM. Write a clean README. This directly echoes your CSIRO Linux/Pi work and is what CDS actually does in production.
